# Little Chemie 8

Interaktives Chemie-Spiel für Klasse 8 nach dem Vorbild von Little Biologie 8.

## Inhalt

- Kombinationsspiel: Stoff → Teilchen → Reaktion → Nachweis → Umweltchemie → Redox
- Keine Säure-/Lauge-Einheit
- Neue Themen: Treibhauseffekt, Treibhausgase, Ozonschicht und Ozonloch
- Missionen mit Fortschritt
- Münzen, Tippkauf und Abzeichen
- Chemie-Blitz als kurzes Quiz
