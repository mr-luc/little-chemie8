# Little Chemie 8

Interaktives Chemie-Spiel für Klasse 8 nach dem Vorbild von Little Biologie 8.

## Inhalt

- Kombinationsspiel: Stoff → Teilchen → Reaktion → Nachweis → Redox
- Missionen mit Fortschritt
- Münzen, Tippkauf und Abzeichen
- Chemie-Blitz als kurzes Quiz
- Chemie-Galaxy als Mini-Spiel

## Start

Öffne `index.html` im Browser oder nutze GitHub Pages, wenn es für dieses Repository aktiviert ist.
